# アプリケーション名	
シン・3日坊主

# アプリケーション概要
* 自分が今までちょっと試してみたかったことのログを残すことができる
* ユーザー同士で応援し合い、新しいチャレンジの情報を得ることができる

# アプリケーションを作成した背景
* 20・30代の若手社会人「勉強しようと思うけど続かなくて・・・・」という悩みに「続けようと思えるものに出会えていないのでは」という答えを。
* 一般的に「続けられない・3日坊主」という言葉を「たくさんのことを試している・新たな経験ができている」という意味合いに変換したかった
